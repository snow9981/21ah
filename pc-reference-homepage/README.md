# PC 참고 이미지 기반 시안

이 폴더는 기존 홈페이지를 덮어쓰지 않고, 사용자가 제공한 참고 이미지의 구도와 분위기를 바탕으로 새로 만든 PC 중심 랜딩 시안입니다.

## 파일

```text
pc-reference-homepage/
├── index.html
├── styles.css
├── script.js
└── README.md
```

## 확인 주소

기존 로컬 서버가 켜져 있다면 아래 주소로 확인할 수 있습니다.

```text
http://127.0.0.1:8765/pc-reference-homepage/index.html
```

## 메모

- 현재 시안은 PC 첫 화면의 브랜드 인상과 레이아웃을 우선으로 만들었습니다.
- 모바일 화면은 별도 전용 레이아웃으로 다시 다듬는 것을 권장합니다.
- 링크와 병원 정보는 기존과 동일하게 placeholder를 사용합니다.
